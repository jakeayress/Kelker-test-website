How to Install

✉
Gmail
  1. Click the gear icon → See all settings
  2. Go to the General tab
  3. Scroll to Signature → create or edit
  4. Click the source icon (<>) in the toolbar
  5. Paste the copied HTML, click OK
  6. Save Changes at the bottom
📧
Outlook
  1. Go to File → Options → Mail → Signatures
  2. Click New, name your signature
  3. In the editor, click the HTML icon or open in browser
  4. Paste the HTML directly into a new email in compose view
  5. Copy the rendered result back into the signature editor
  6. Click OK to save
🍎
Apple Mail
  1. Go to Mail → Preferences → Signatures
  2. Create a new signature and type placeholder text
  3. Quit Mail, then open the signature file in ~/Library/Mail/V10/MailData/Signatures/
  4. Replace the file contents with the copied HTML
  5. Reopen Mail — your signature is ready
